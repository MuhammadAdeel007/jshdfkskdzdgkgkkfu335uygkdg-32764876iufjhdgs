# LOCATION SYSTEM

Programmatic page architecture for every geographic tier.

## Tiers

* State (50 pages)
* County (~3,143 pages)
* City (~19,500 pages)
* Metro (derived, optional)

## Page Composition

Every location page contains:

* Local Overview
* Regional Insights
* Market Information
* Common Service Categories
* Common Issues
* Service Availability indicator
* Cost Factors (regional pricing drivers)
* Local FAQs
* Related Locations
* Related Services
* Conversion-safe CTA sections
* Compliance Disclaimer

## Prohibited Content

* Office addresses
* Embedded maps
* LocalBusiness schema
* "Near me" claims
* Office hours
* Service-area polygons

## Schema

* WebPage
* BreadcrumbList
* FAQPage (only where FAQ block exists)
* Organization (sitewide)
* No LocalBusiness, no address markup
