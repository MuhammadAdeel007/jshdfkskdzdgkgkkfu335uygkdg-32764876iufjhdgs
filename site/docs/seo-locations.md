# TECHNICAL SEO — LOCATION PAGES

## Indexability

* All location pages indexable, follow
* Canonical self-references
* XML sitemap segmented by tier
* No noindex on generated pages

## Performance

* Static HTML preferred over client-rendered
* Critical CSS inlined
* Images lazy-loaded (location pages use zero hero images by default)
* Target LCP < 2.5s on 4G mobile

## Structured Data

* BreadcrumbList on every page
* FAQPage where FAQ block is present
* Organization sitewide
* No LocalBusiness markup (prohibited)

## Metadata

* Unique title per page
* Unique meta description per page
* OG title/description derived from location name
* No duplicate descriptions across tiers

## Internal Linking

* Every state links to counties
* Every county links to cities + state
* Every city links to county + state
* All tiers link to relevant services
* Related Locations + Related Services blocks auto-generated
